  • IDE (Ingeniería De Equipos) ha un'esperienza di più di 40 anni in realizzazione di ingegneria d'impianti industriali e particolarmente nel progettare di equipaggiamento, processi e disegno meccanico. * * * * Il creatore dell'IDE * * * *
    Per tutto tipo d'equipaggiamento IDE può eseguire l'ingegneria completa: specificazioni, monografie del calcolo, disegni; così come tutti i compiti riferirono all'procurement, trasporto, montaggio e start-up.
  • Abbiamo utilizzato i nostri inizi, per la realizzazione di ogni fase dell'ingegneria delle apparecchiature, dei fogli di calcolo che abbiamo sviluppato. Oggi abbiamo anche le software riconosciuto e una conoscenza profonda dell'internazionalmente norme usate (ASTM, ASME, TEMA, BS, ANSI, API, AD-MERKBLATTER ) facendo il lavoro molto più rapido, affidabile e economico.
  • Noi eseguiamo anche la valutazione dell'integrità strutturale e di attitudine per il servizio delle attrezzature menzionate secondo API 579, API 510, API 572, API 580 ed BS 7910. Anche l'analisi di tensioni col metodo di elemento finiti (FEM / FEA).
  • Calcolo e progetto del sistema di tubazioni, analizzando flessibilità, sollecitazioni localizzate e fatica.
