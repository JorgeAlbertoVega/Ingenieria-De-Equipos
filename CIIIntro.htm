Caesar II Tutorial Introduction :
CAESAR II is a pipeline stress analysis software that uses beam theory to evaluate pipeline systems to numerous international standards. CAESAR II is not a finite element analysis (FEA) software, but rather uses a linear model constructed of elements connected by nodes. This tutorial describes how exercises serve to introduce the CAESAR II program and demonstrate various modeling and analysis methods to evaluate and correct pipeline systems.
